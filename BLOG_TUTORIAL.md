# GeoFill：一款基于 IP 定位的智能注册信息生成浏览器插件

> 🌍 告别繁琐的表单填写，让注册变得轻松愉快！

## 📖 前言

你是否在测试网站、注册账号时，厌倦了反复填写那些冗长的表单信息？**名字、邮箱、电话、地址...**层层叠叠的输入框让人头疼。

今天给大家分享一款我开发的浏览器插件——**GeoFill 智能注册助手**。它能根据你当前的 IP 地理位置，**智能生成真实格式的注册信息**并**一键填写表单**，大大提升测试和开发效率。

## ✨ 核心功能亮点

### 🌍 智能 IP 定位

- 自动检测当前 IP 的国家、城市、州/省信息
- 支持手动刷新 IP 位置
- 可手动切换到其他国家，信息会自动匹配更新

### 📝 真实格式信息生成

| 字段 | 智能特性 |
|------|----------|
| 姓名 | 根据国家生成对应语言的姓名（中国生成中文名，日本生成日文名等） |
| 性别 | 随机生成，支持表单 radio 按钮 |
| 生日 | 随机生成 18-55 岁之间的日期 |
| 用户名 | 基于姓名生成多种风格 |
| 邮箱 | 支持 Gmail、Outlook 等多种后缀，也可自定义 |
| 密码 | 12 位强密码（包含大小写、数字、符号） |
| 电话 | 各国真实号段和格式（如美国 +1、中国 +86） |
| 地址 | 城市、州/省、邮编保持真实对应关系 |

### 🎯 表单填写增强

- 智能识别多种表单字段格式
- 完美兼容 **React、Vue** 等现代前端框架
- 自动识别 fullname 字段并拆分填写
- 支持密码和确认密码双填

### 🎨 贴心的界面设计

- 支持 **深色/浅色主题** 切换
- 每个字段都可 **单独刷新** 或 **锁定**
- 一键复制任意字段
- 操作成功有 Toast 提示

## 🚀 安装教程

### 方法一：Chrome / Edge 浏览器

1. **下载插件源码**
   - 从 [GitHub 仓库](https://github.com/your-repo/GeoFill) 下载并解压
   - 或直接下载 `GeoFill.zip` 并解压到任意文件夹

2. **打开扩展程序页面**
   - Chrome 浏览器：地址栏输入 `chrome://extensions/`
   - Edge 浏览器：地址栏输入 `edge://extensions/`

3. **开启开发者模式**
   - 点击页面右上角的「**开发者模式**」开关

4. **加载插件**
   - 点击「**加载已解压的扩展程序**」按钮
   - 选择解压后的插件文件夹

5. **完成安装**
   - 插件图标会出现在浏览器工具栏
   - 如果没有显示，点击工具栏的拼图图标 🧩，将 GeoFill 固定到工具栏

### 方法二：Firefox 浏览器

1. 打开地址 `about:debugging#/runtime/this-firefox`
2. 点击「**临时加载附加组件**」
3. 选择项目中的 `manifest.json` 文件

> ⚠️ Firefox 临时加载的插件在浏览器重启后会失效，需要重新加载

## 📖 使用教程

### 基本使用流程

1. **打开目标网站**
   - 导航到任意需要注册的网站页面

2. **点击插件图标**
   - 点击浏览器工具栏中的 GeoFill 🌍 图标
   - 弹出面板会自动检测你的 IP 位置并生成匹配信息

3. **查看生成的信息**
   - 顶部显示当前检测到的 IP 位置
   - 下方列出所有生成的注册信息字段

4. **调整信息（可选）**
   - 🔄 点击刷新按钮可重新生成单个字段
   - 🔒 点击锁定按钮可保护该字段不被刷新
   - 📋 点击复制按钮可快速复制到剪贴板
   - 也可以直接在输入框中手动修改

5. **一键填写表单**
   - 确认信息无误后，点击底部的「**✍️ 填写表单**」按钮
   - 插件会自动将信息填入页面表单中

### 进阶功能

#### 🔄 刷新 IP 位置

如果你使用了 VPN 或代理更换了 IP，可以点击 IP 信息旁边的 🔄 按钮手动刷新位置检测。

#### 🌍 手动切换国家

在「国家」下拉菜单中选择其他国家后，所有相关信息（姓名、电话、地址等）会自动根据新国家重新生成。

#### 📧 自定义邮箱后缀

在「后缀」下拉菜单中可以选择：
- `gmail.com`（默认）
- `outlook.com`
- `yahoo.com`
- `hotmail.com`
- `icloud.com`
- `protonmail.com`
- 或选择「✏️ 自定义」输入任意域名

#### 🔒 锁定字段

当你对某个生成的值很满意时，点击 🔓 按钮可以锁定该字段。锁定后：
- 点击「🔄 重新生成」不会改变该字段
- 切换国家也不会覆盖该字段

#### 🌙 主题切换

点击右上角的 🌙/☀️ 按钮可以切换深色/浅色主题。

## 📁 技术实现

### 项目结构

```
GeoFill/
├── manifest.json       # 插件配置清单 (Manifest V3)
├── popup/
│   ├── popup.html      # 弹出面板界面
│   ├── popup.css       # 样式文件
│   └── popup.js        # 面板交互逻辑
├── scripts/
│   ├── generators.js   # 信息生成器核心模块
│   └── content.js      # 表单填写内容脚本
└── icons/              # 插件图标
```

### 技术栈

- **Manifest V3**：使用最新的 Chrome 扩展规范
- **原生 JavaScript**：无框架依赖，轻量高效
- **Chrome Storage API**：持久化存储用户设置和锁定状态
- **Content Script**：实现跨页面表单填写
- **IP 定位 API**：使用 ipapi.co 和 ip-api.com 双重定位

### 核心技术点

1. **框架兼容性**
   - 使用 `Object.getOwnPropertyDescriptor` 获取原始 setter
   - 通过 `inputEvent` 和 `change` 事件触发框架状态更新
   - 完美支持 React、Vue 等现代前端框架

2. **智能字段匹配**
   - 多种选择器匹配策略
   - 支持 name、id、placeholder、autocomplete 等属性
   - 特殊处理 fullname 拆分和密码确认字段

3. **数据缓存**
   - 生成的信息会自动缓存
   - 下次打开弹窗时无需重新生成
   - 支持版本控制，更新插件时自动清除旧缓存

## 🔧 常见问题

### Q: 为什么有些字段没有被填写？

**A:** 可能是网站使用了非标准的表单字段名称。插件会尽可能匹配常见的字段选择器，但某些高度定制化的表单可能无法完全覆盖。

### Q: 填写后表单验证报错？

**A:** 某些网站可能有特殊的验证规则。你可以：
1. 手动修改生成的信息后再填写
2. 锁定正确的字段，重新生成其他字段

### Q: IP 位置检测失败？

**A:** 可能是网络问题或 IP 定位服务暂时不可用。插件会自动降级使用默认值（美国），你也可以手动选择国家。

### Q: 支持移动端吗？

**A:** 目前仅支持桌面版 Chrome、Edge 和 Firefox 浏览器。
